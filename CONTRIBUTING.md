# Contributing

Patches can be sent via email to `~the-commits/tjenamors-se-tailscale@lists.sr.ht`
or pull requests on [sourcehut](https://git.sr.ht/~the-commits/tjenamors-se-tailscale).

The [GitHub mirror](https://github.com/the-commits/tjenamors-se-tailscale) is
read-only — please do not open issues or PRs there. Use sourcehut instead.

All commits must be GPG-signed.
